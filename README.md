# Antigravity 16-bit 核心防禦 (Core Defense)

這是一款採用 **Next.js**, **TypeScript** 與 **HTML5 Canvas** 原生開發的 16-bit 像素風網頁塔防遊戲。

## 遊戲特色 (Features)

- **無盡模式 (Endless Mode)**：沒有過關限制！抵禦一波接一波越來越強的機甲敵軍，挑戰您的極限生存波次。
- **動態尋路系統 (A* Pathfinding)**：敵人會即時尋找前往主堡的最短路徑。玩家可以自由放置城牆與防禦塔來改變敵軍的行進路線，打造專屬的防禦迷宮。
- **多樣化防禦工事**：
  - **大型城牆 (WALL)**：堅固的 `2x1` 金屬牆壁，可按右鍵旋轉方向，用來引導敵軍。
  - **自動砲台 (TURRET)**：高射速的實體子彈防禦塔。
  - **雷射砲 (LASER)**：發射高壓光束並附帶 2 秒「停滯」控場效果。
  - **飛彈塔 (MISSILE)**：發射具追蹤能力的飛彈，擊中時產生巨大爆炸，對周圍敵人造成「範圍傷害 (AoE)」。
- **線上戰績排行榜**：當主堡被摧毀時，可輸入您的名字並將「分數」與「生存波次」儲存至後端資料庫記錄。
- **便捷操作**：支援「滑鼠滾輪」快速切換建築，內建一鍵「RESTART」按鍵與「空白鍵拆除建築」功能(開發中)。

## 開發技術棧 (Tech Stack)

- **框架**：Next.js (App Router), React
- **語言**：TypeScript, HTML5 Canvas API
- **樣式**：Tailwind CSS (輔助排版) / Vanilla CSS
- **資料儲存**：Next.js API Route + 本地 `data.json`

## 如何在本地端執行 (How to Run Locally)

1. 確認您已安裝 [Node.js](https://nodejs.org/)。
2. 複製專案後，在專案根目錄下執行以下指令安裝依賴：
   ```bash
   npm install
   ```
3. 啟動開發伺服器：
   ```bash
   npm run dev
   ```
4. 開啟瀏覽器並前往 `http://localhost:3000` 即可開始遊玩！

## 玩法說明 (How to Play)

- 目標：保護畫面右側發光的「能量核心」。
- 金錢：隨著時間推移會自動獲得金幣，擊敗敵人也會掉落大量金幣與分數。
- 建築：使用滑鼠點擊放置防禦塔。利用城牆把敵人引導到防禦塔火力最密集的地方！
