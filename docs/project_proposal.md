# 遊戲專案企劃書：Antigravity 16-bit 核心防禦

## 1. 專案總覽 (Project Overview)
**專案名稱**：Antigravity Core Defense (反重力核心防禦)  
**遊戲類型**：塔防遊戲 (Tower Defense) / 陣地防禦 (Base Defense)  
**美術風格**：16-bit 復古像素風 (SNES/Sega Genesis 時代風格)，帶有金屬、光影與科幻元素。  
**開發技術**：Next.js (App Router), TypeScript, HTML5 Canvas API, Vanilla CSS。  
**平台**：Web 瀏覽器 (PC 端為主，依賴滑鼠滾輪與點擊操作)。

## 2. 遊戲核心理念 (Core Concept)
本作靈感源自「Antigravity (反重力)」與傳統陣地戰的結合。有別於傳統塔防沿著固定路線行進的怪獸，本遊戲採用**「開放式網格地圖」與「動態 A* 尋路演算法」**。
玩家扮演星際基地的指揮官，必須在廣闊的星艦甲板上，自由利用城牆與各種防禦塔，構築出複雜的防禦迷宮，誓死保衛位於地圖最右側的「能量核心 (Core)」，抵禦從左側不斷湧入的機甲敵軍。

## 3. 核心遊戲機制 (Gameplay Mechanics)

### 3.1 無盡模式 (Endless Mode) 與動態尋路
- **目標**：保護地圖最右側的能量核心 (20 HP)。敵人碰到核心會扣除 1 HP，核心歸零即 GAME OVER。
- **無盡挑戰**：沒有固定的勝利條件，波次無限增加，敵人數量與血量會隨之成長，玩家必須盡可能活得越久越好。
- **敵人生成**：敵人統一由地圖最左側隨機高度出現，並利用 A* 演算法即時尋找前往核心的最短路徑。
- **自由佈陣 (防堵死機制)**：玩家可以自由放置城牆與防禦塔來改變敵人的行進路線（迷宮化）。系統內建防呆機制，若玩家試圖完全封死核心，系統將以紅色預覽框提示並禁止放置，確保遊戲順利進行。

### 3.2 經濟與計分排行榜系統
- **分數與金幣**：擊敗敵人可同時獲得金幣 (Gold) 與分數 (SCORE)，敵人的強度越高，給予的分數與金幣越豐厚。
- **被動收入**：隨著時間推移，玩家每秒會自動獲得 5 枚金幣，確保在防守空檔也能持續佈置陣地。
- **歷史排行榜**：當主堡被摧毀 (Game Over) 時，畫面會浮現結算成績單與簽名檔。玩家可以輸入自己的名字，系統會自動將「名字、最終分數、生存波次」記錄到後端的 `data.json` 保存。

### 3.3 防禦建築系統
玩家可透過**滑鼠滾輪**快速切換三種防禦建築，點擊左鍵花費金幣建造：

1. **大型城牆 (WALL) - 10G**
   - **特性**：佔地 `2x1` 格子的大型金屬牆，擁有極高的血量 (400 HP)。
   - **功能**：會被尋路演算法視為「絕對障礙物」，專門用來改變敵人行進路線，且敵人必須繞道而行。
2. **自動砲台 (TURRET) - 50G**
   - **特性**：佔地 `1x1` 格子，血量中等 (150 HP)。
   - **功能**：自動鎖定範圍內最近的敵人發射高射速的實體子彈。**不阻擋敵人行進**，敵人可穿梭其間。
3. **雷射砲 (LASER) - 100G**
   - **特性**：佔地 `1x1` 格子，血量較低 (120 HP)。
   - **功能**：發射瞬間命中的雷射光束，除造成傷害外，還能讓敵人陷入「停滯狀態」2 秒，是極為強大的控場武器。**不阻擋敵人行進**。
4. **飛彈塔 (MISSILE) - 150G**
   - **特性**：佔地 `1x1` 格子，血量較高 (200 HP)，擁有專屬的發射井外觀。
   - **功能**：發射具有追蹤能力的橘光飛彈。當命中目標時會產生巨大的爆炸光暈，對半徑 2 格內的所有敵人同時造成巨量的**範圍傷害 (AoE)**。雖然射速極慢，但在應付大量敵群時具備毀滅性的壓制力。**不阻擋敵人行進**。

### 3.4 敵人的反擊
有別於傳統塔防，本遊戲的建築是有**耐久度 (HP)** 的。
敵人在尋路的過程中，若偵測到其周圍（或其踩踏的格子）有玩家的建築物，他們會**主動停下來攻擊建築**。若建築被摧毀，原本被阻擋的路線將會重新開放，迫使所有敵人重新規劃路徑，帶來極大的動態戰局變化。

## 4. 視覺與美術設計 (Art & UI Design)

- **16-bit 像素科幻風**：
  - **背景**：深藍色宇宙背景搭配星艦金屬甲板網格，網格帶有十字接點細節。
  - **敵人**：立體幾何機甲造型，擁有駕駛艙高光、機體陰影與動態閃爍的「推進器火焰」動畫，並會在受到雷射攻擊時變為藍色冰凍狀態。
  - **建築**：城牆帶有裝甲接縫與立體光影；砲台具備可旋轉的砲管與底座陰影；雷射砲則具有如呼吸般閃爍的能量光暈特效。
  - **UI 介面**：使用 8-bit Google Font (`Press Start 2P`)，搭配具有 Neon 光暈效果的選單與血條，營造濃厚的 Retro Arcade 氛圍。

## 5. 技術架構 (Technical Architecture)

- **框架**：Next.js App Router 作為 React 的宿主環境，並提供一個輕量級的 API Route (`/api/save`) 用於保存玩家通關資料。
- **渲染引擎**：完全依賴原生的 HTML5 `<canvas>` 與 `CanvasRenderingContext2D`。不使用任何外部遊戲引擎，確保極致的輕量化與效能。
- **遊戲主迴圈**：使用 `requestAnimationFrame` 實現 60 FPS 的 Game Loop。透過 `dt (delta time)` 來控制所有單位的移動速度與冷卻時間，確保在不同更新率的螢幕上擁有相同的遊戲體驗。
- **尋路演算法**：客製化的 A* 尋路演算法，以曼哈頓距離 (Manhattan Distance) 作為 Heuristic，並動態考量建築物的佔地大小 (`w, h`) 與是否阻擋 (`isObstacle`)。

## 6. 未來擴充計畫 (Future Roadmaps)
1. **關卡與無盡模式**：引入更多地圖樣式（如地形障礙），並實作無限波次 (Endless Mode) 與排行榜。
2. **多樣化敵軍**：加入高移速的「刺客型」、血量極厚的「坦克型」，甚至是會遠程攻擊防禦塔的「砲兵型」敵人。
3. **音效與配樂**：導入 16-bit Chiptune 復古電子配樂與爆炸、發射雷射的音效，提升沉浸感。
4. **存檔系統完善**：將 `data.json` 升級為資料庫，實作玩家登入與成就系統。
